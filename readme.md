# 💎 Kiyohara-Style Small Cap Hunter Pro (清原式 スクリーニングツール)

## 概要
「割安小型成長株」の投資理論（清原達郎氏のメソッド）に基づき、財務データから有望な銘柄を自動発掘・スコアリングする分析ツールです。
PythonとStreamlitを用いて開発され、財務健全性（ネットキャッシュ比率）、割安性（PBR）、成長性などを多角的に評価します。

## 特徴
* **科学的なスコアリング:** 「現金比率」「PBR」「営業CF」など複数のKPIに基づき、感情を排した定量評価を行います。
* **青天井のS級判定:** 異常な割安株（ネットネット株など）を検出するため、スコア上限を撤廃した独自のアルゴリズムを実装。
* **リスク回避機能:** 営業キャッシュフロー赤字などの「罠銘柄」を検知し、スコアを自動減点する安全装置を搭載。
* **インタラクティブなUI:** Streamlitによる直感的な操作と、Plotlyによるレーダーチャート可視化。

## 使用技術
* Python 3.x
* Streamlit (Frontend)
* Pandas (Data Processing)
* Plotly (Visualization)
* yfinance (Market Data)
